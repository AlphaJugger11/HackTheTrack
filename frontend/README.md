# GR Cup Analytics Platform - Frontend

React + TypeScript + Vite frontend for the GR Cup Analytics Platform.

## Setup

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

Frontend will run on http://localhost:5173

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- Plotly.js (charts)
- D3.js (track visualization)
- Axios (API calls)
